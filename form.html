Adicionando Formulários

Para que serve um formulário na web?

Formulário é uma das elementos mais úteis da web, ele está por todas as partes. Para um cadastro simple de login, cadastro uma dado em site, pedir um iFodd... São várias as opções.

E como funciona um formulário? Lembra da lei da atração? Tudo o que vai volta, com os formulários funciona da mesma forma, você enviar algo para uma parte, e depois a outra parte de devolver o que você pediu.

Para isso que serve um atributo chamado action, ele especifica para onde os dados devem ir quando você apertar no botão enviar. Para entender melhor, precisamos estudar um pouco de Cliente e Servidor

Elemento Input e Label

O elemento mais comum no form é o input, que significa entrada, e você pode usa-lo em mais de 20 diferente tipos de entradas, por isso você precisa mudar apenas o atributo type. veja alguns exemplos abaixo:

O input não possui tag de fechamento. E como mencionei, o atributo mais importante é o type, veja abaixo mais tipos de entrada mais importante:

  • text - para textos
  • password - para senhas
  • color - para selecionar cor
  • number - para números
  • submit - botão de enviar
  • email para email
  • date - para selecionar datas
